# Moo Docs
Documentation and notes for cows

## Usage

Clone repo<br />
```
git clone https://github.com/EdyTheCow/docs.edy.io.git
```

Start development server<br />
```
docker-compose up watch
```

Build docs<br />
```
docker-compose up build
```
